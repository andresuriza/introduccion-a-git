# Portón Automatico

## Descripción

Sistema que al presionar un botón desde un control, se abre el portón si está cerrado o se cierra si está abierto.

## Usuario Objetivo

Casas, edificios y distintos lugares donde se ocupe acceder con un vehiculo.

## Entradas

- Sensor infrarojo: recibe señal de botón del control
- Sensor de proximidad que evita cerrar si algún objeto o persona se encuentra en medio del portón.
